# IoT-Dashboard

## Environment
node.js 20.x

### Install dependencies
```
$ npm install
```
### Build project
```
$ npm run build
```
### Start project
```
$ npm run start
```
### Start project as developer version
```
$ npm run dev
```

## setting in env file
```
NEXT_PUBLIC_APP_SERVER_URL={ server url }
NEXT_PUBLIC_GOOGLE_MAPS_API_KEY={ google map api key }
```
